## Using

Preconditions:

```
sudo apt-get install libbz2-dev
```

Compile:

```
make
```
Run:

```
make run o=origin_dir d=destiny_dir.bz2.tar
```

Check:

```
make check o=origin_dir d=destiny_dir
```
